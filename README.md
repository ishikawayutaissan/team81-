# お問い合わせフォーム
 お問い合わせフォームです。質問したい内容をメールで送ることができます。<br/>
 また管理者画面でお問い合わせの検索、ユーザーの登録、編集を行えます。

## 使用言語
Laravel <br/>
(bootstrapも使用しています。)
　
## 機能
ログイン<br/>
メール送受信<br/>
検索<br/>
会員登録<br/>
登録情報の編集<br/>

  
## 注意事項
会員登録者の一覧がありますが、編集、削除をしないで下さい。

## インストール方法について

$ git clone    <br/>
$ cd  <br/>
$ sh setup.sh<br/>
$ ~do anything~<br/>
